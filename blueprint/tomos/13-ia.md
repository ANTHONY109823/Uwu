# 📖 Tomo XIII — IA (Futuro)

> Funciones planificadas para v2.0+. No implementar en v1.0.

## Visión

Integrar IA para reducir la fricción creativa: ayudar al usuario a escribir mensajes, elegir canciones y personalizar experiencias sin perder autenticidad.

---

## Funciones propuestas

### 1. Generador de mensajes

**Input:** ocasión, nombre destinatario, tono (romántico, divertido, formal)  
**Output:** 3 opciones de mensaje para elegir o editar

```
Prompt: "Escribe un mensaje romántico para el cumpleaños de Mariana, 
tono dulce y divertido, máximo 200 palabras, en español."
```

**UI:** botón "✨ Sugerir mensaje" en editor → modal con 3 opciones.

---

### 2. Sugeridor de canciones

**Input:** ocasión, mood, idioma  
**Output:** 5 canciones sugeridas con artista

Integración con Spotify API para preview de 30s.

---

### 3. Asistente de personalización

Chat inline en el editor:

```
Usuario: "Quiero algo más divertido"
IA: "¡Perfecto! Te sugiero cambiar el mensaje a algo con humor 
     y agregar emojis de globos 🎈. ¿Quieres que lo reescriba?"
```

---

### 4. Generación de imágenes

**Input:** descripción textual  
**Output:** imagen decorativa para la dedicatoria

Usar API de generación de imágenes. Solo tier Exclusiva.

---

### 5. Traducción automática

Mensaje en español → traducir a inglés/portugués para destinatarios internacionales.

---

## Arquitectura propuesta

```
Frontend → POST /api/ai/generate-message
                ↓
         AI Service (NestJS module)
                ↓
         OpenAI API / Anthropic API
                ↓
         Rate limit + cost tracking
```

---

## Consideraciones

| Aspecto | Decisión |
|---------|----------|
| Privacidad | No almacenar prompts con datos personales |
| Costo | Limitar a 3 generaciones por dedicatoria |
| Calidad | Siempre editable por el usuario |
| Tier | Premium: 3 usos, Exclusiva: ilimitado |
| Fallback | Si IA falla, editor funciona normal |

---

## No hacer con IA

- No generar plantillas completas automáticamente (pierde identidad UWU)
- No reemplazar el editor visual
- No enviar datos de pago a modelos de IA
- No auto-publicar sin confirmación del usuario
