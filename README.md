## Stack

**Frontend:** Next.js 15 · React · TypeScript · Tailwind CSS · shadcn/ui · Framer Motion → **Vercel Pro**
**Backend:** NestJS 11 · Prisma · PostgreSQL · Redis · BullMQ → **Railway Pro**
**Media:** Cloudflare R2 · **Pagos:** MercadoPago + Stripe · **Monorepo:** Turborepo + pnpm

## El flujo core

```
Elegir plantilla → Personalizar → Preview en vivo → Pagar → Página generada → Enlace + QR → Compartir 💝
```
